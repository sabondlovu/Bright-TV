# Bright-TV Analysis 
This is for Bright TV analysis 

Objectives for this analysis: to help the CEO have a better understanding of TV viewership. So they can gain good insights on how to improve the channels performance. The main outcome will be 
- Understanding the trends of the user 
- Understanding the viewership 
- Have recommendations for increasing viewership on those channels that are not performing well
-

Main Viewership Trends 
-Most of the viewers are Young Adults between 18 and 34, and they are located in Gauteng. They have the highest segments 
- Their content consumption is Sports and Music
- Then we see a slump when it comes to Monday, which shows a 30%-40% drop-ff in activity every Monday 
- Great viewership is at night, which is mainly consumed by Adults, while the kids' content is stable in the morning to afternoon.

- Recommendations
- We can use our weekend sports and music into Monday must watch content and using Gauteng base to drive referrals
- Use some viewers as word of mouth, make viewers talk about sports that are coming on a Monday. People trust people who are like them.
